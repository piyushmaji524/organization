# Contributing to Sarak Youth Development Council Website

## Code of Conduct
Please be respectful and constructive in all interactions.

## How to Contribute

### 1. Fork & Clone
```bash
git clone https://github.com/YOUR_USERNAME/sarak-youth.git
cd sarak-youth
```

### 2. Create a Branch
```bash
git checkout -b feature/your-feature-name
# or
git checkout -b fix/your-bug-fix
```

### 3. Make Changes
- Follow the existing code style
- Add comments for complex logic
- Test your changes locally

### 4. Commit with Clear Messages
```bash
git commit -m "feat: add new feature" 
# or
git commit -m "fix: resolve issue with X"
```

### 5. Push & Create Pull Request
```bash
git push origin feature/your-feature-name
```

## Development Setup
See `SETUP.md` for local development instructions.

## Reporting Issues
- Use GitHub Issues for bug reports and feature requests
- Include screenshots/logs if applicable
- Describe the expected vs actual behavior

## Code Style
- Use PSR-12 PHP coding standards
- Use semantic HTML5
- Use camelCase for JavaScript variables
- Add meaningful comments for non-obvious code

Thank you for contributing! 🙏
