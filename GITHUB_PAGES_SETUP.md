# ⚙️ Enable GitHub Pages - Step by Step

## 🎯 Goal
Enable your custom landing page with CSS at: `https://piyushmaji524.github.io/organization`

---

## 📋 Step-by-Step Instructions

### Step 1: Go to Repository Settings
```
1. Open: https://github.com/piyushmaji524/organization
2. Click on "Settings" tab (top right)
3. Scroll down to "Code and automation" section
```

### Step 2: Navigate to Pages
```
In the left sidebar:
- Find "Pages" option
- Click on it
```

### Step 3: Configure GitHub Pages
```
Under "Build and deployment":

1. SOURCE
   - Select: "Deploy from a branch"

2. BRANCH
   - Select: "main"

3. FOLDER
   - Select: "/ (root)" OR "/docs" 
   - Choose "/docs" if you want docs/index.html

4. Click "Save"
```

### Step 4: Wait & Verify
```
- GitHub will build your site (1-2 minutes)
- You'll see: "Your site is live at https://piyushmaji524.github.io/organization/"
- Your custom landing page will be live!
```

## ❓ Troubleshooting

### Landing page not showing?
- ✅ Check that Pages is enabled (Settings > Pages)
- ✅ Wait 2-3 minutes for deployment
- ✅ Clear browser cache (Ctrl+Shift+Delete)
- ✅ Try incognito/private browsing

### Still seeing README instead of landing page?
- ✅ Check folder is set to `/docs` in Pages settings
- ✅ Verify `docs/index.html` file exists
- ✅ Make sure no `index.md` in `/docs` folder

### CSS not loading?
- ✅ CSS is inline in HTML file (no external files)
- ✅ Try hard refresh: `Ctrl+Shift+R`
- ✅ Check browser console for errors (F12)
